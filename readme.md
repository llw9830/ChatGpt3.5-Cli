## development

```bash
go build

或者下载.exe文件直接运行
```

```bash
-- 1、设置apiKey
go-gpt3-cli.exe key set <apiKey>
-- 2、设置对话角色
go-gpt3-cli.exe sys set <以私人智能助手身份和我说话>
-- 3、运行
go-gpt3-cli.exe run
```
